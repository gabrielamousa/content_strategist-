E se você pudesse fazer X? Tendência que está mudando Y

Template reproduzível para qualquer tema: gancho em pergunta + prova social + nome/definição + exemplos (global e local) + mecanismo + fechamento com CTA de marcação.

Post original https://www.instagram.com/p/DGK45rmtVJw/

Resumo

Story + Educação + Tendência
Ideia central: abrir um “e se…?” que projeta o público num desejo, provar que é real (mídia/prints/dados), nomear a tendência (dá linguagem), mostrar 2 exemplos (fora + aqui) e fechar com mecanismo simples + pergunta + marcação.
Objetivo do post
Gerar identificação + compartilhamentos
Retenção
Open loop + prova + exemplos + mecanismo
Tom
Aspiracional, realista, curioso
Regra de ouro
1 ideia por slide (máx. 2 frases)

Roteiro (copiável)

Texto slide a slide
Dica: personalize os campos em colchetes para qualquer nicho.

Estrutura

Sequência e intenção
  1. Gancho em pergunta (curiosidade + projeção pessoal) + seta de continuidade.
  2. Prova social (prints, dados, “isso existe”).
  3. Nome + definição (dar vocabulário e clareza em 2–3 frases).
  4. Exemplo global (autoridade/escala).
  5. Exemplo local (viabilidade/“dá pra fazer aqui”).
  6. Mecanismo (3 pontos: como funciona, sem jargão).
  7. Fechamento + pergunta + CTA (marcar amigos + salvar/compartilhar).
Por que funciona: alterna emoção (sonho) → realidade (prova) → entendimento (definição) → evidência (exemplos) → lógica (mecanismo) → ação (CTA).

Elementos visuais

High-tech + legível
  • Base: foto em tela cheia (aspiracional) alternada com colagem de prints (prova).
  • Legibilidade: use box semitransparente escuro (55–70% opacidade) quando o fundo estiver “poluído”.
  • Consistência: texto dominante no centro (40–60% da altura). Uma palavra em destaque por slide-chave.
  • Ritmo: aspire (1) → prova (2) → conceito (3) → casos (4–5) → mecanismo (6) → chamada humana (7).
  • Destaques: aplique um gradiente (verde→azul→laranja) em palavras-âncora, linhas ou sublinhados.
TIPOGRAFIA REPRODUZÍVEL H1 (gancho): grande, poucas linhas, estilo editorial (serif/itálico se quiser “sonho”). H2 (conceito/CTA): sans, grande, cor destaque. Body: sans, branco, 2–3 frases curtas. Micro-CTA: pequeno + setas “>>>”.

Fechamentos (slide 7)

CTA

Engajamento de baixa fricção
  • CTA de avanço (slide 1): “Vem ver como funciona >>>”
  • CTA de salvar (slide 5 ou 6): “Salva pra pesquisar depois.”
  • CTA de comentário (slide 7): pergunta simples com “sim/não + por quê”.
  • CTA de marcação (slide 7): “Marca aqui quem toparia isso com você.”
  • CTA de compartilhamento: “Manda pra alguém que vive falando de [tema].”

Variações

Mesma força, outros temas
Como adaptar sem perder impacto: mantenha a mesma “espinha dorsal” (pergunta → prova → nome → exemplos → mecanismo → CTA) e troque apenas os blocos entre colchetes.
  • Se você não tem prints/manchetes: use “prova leve” (Google Trends, dado público, gráfico simples, depoimento anônimo, antes/depois, ou 2 prints de comentários).
  • Se não existe exemplo global: use “exemplo de outro setor” (analogia) e, no slide 5, traga 1 exemplo real próximo.
  • Se o tema é polêmico: troque “isso é o futuro” por “isso é uma alternativa” e faça pergunta neutra no final.
  • Se o objetivo é venda: no slide 6, inclua “o que ninguém te conta” + no slide 7: “Quer o passo a passo? comenta ‘QUERO’”.
  • Se o objetivo é autoridade: adicione no slide 2: “o que eu observei em [experiência]” e, na legenda, 3 aprendizados numerados.
BANCO DE SUBSTITUIÇÕES (X / Y) • Carreira: X = trabalhar melhor / Y = burnout/solidão • Finanças: X = gastar menos sem sofrer / Y = inflação/impulsos • Saúde: X = ter energia / Y = rotina caótica • Estudos: X = aprender rápido / Y = falta de foco • Negócios: X = vender com consistência / Y = sazonalidade • Lifestyle: X = viver com mais leveza / Y = excesso de coisas
