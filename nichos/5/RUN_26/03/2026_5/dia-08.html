Carrossel vitrine: seus melhores projetos do ano

Modelo reproduzível de curadoria/portfólio: capa que abre loop + mosaicos 3×2 de miniaturas por projeto + fechamento com CTA de DM. Feito para gerar retenção, prova social e demanda de orçamento.

Resumo

O que este post faz: vende repertório + consistência + capacidade de adaptar estilo para clientes diferentes (prova + autoridade leve).

Objetivo
Portfólio rápido → DM/orçamento
Mecânica de retenção
Padrão repetível + variedade visual
Tempo de produção
60–90 min (com assets prontos)
Quantidade ideal
8–12 slides
Curadoria Portfólio Mosaico 3×2 Branding consistente

Estrutura (sequência fixa)

  • Slide 1 — Capa (hook): “Meus [FORMATOS] favoritos de [ANO]” + palavra-âncora grande + elemento de cor + sua foto/recorte.
  • Slides 2–N-1 — Mosaicos: topo “Carrossel para [Cliente/Projeto]” + grade 3×2 com 6 miniaturas do projeto.
  • Último slide — Fechamento: pergunta (“qual foi seu favorito?”) + CTA principal (DM com palavra) + CTA secundário (salvar).

Regra de ouro: repete o layout para dar leitura rápida e muda o visual de cada projeto para manter novidade.

Roteiro (reproduzível)

Pré-produção (10–15 min)

  • Selecione 6–10 projetos com variedade (cor, tipografia, nicho, objetivo).
  • Para cada projeto, escolha 6 telas (as mais fortes/representativas).
  • Defina a palavra-âncora: “favoritos”, “melhores”, “mais pedidos”, “top”.

Montagem (40–60 min)

  • Crie a capa com hierarquia em camadas (serif + sans + script).
  • Monte 1 slide-mãe de mosaico 3×2 e duplique para o resto.
  • Em cada slide: troque só título + miniaturas (padrão constante).
  • Finalize com slide de CTA em 2 passos (DM + comentário/salvar).

Revisão final (10 min)

  • Cheque legibilidade em 30% de zoom (mobile).
  • Consistência de margens/alinhamentos entre slides.
  • Miniaturas com borda/sombra leve para separar do fundo.

Slides sugeridos (1..10)

  • 1 — Capa: “Meus carrosséis favoritos de 2026” + sua foto + acento forte.
  • 2 — Projeto 01: “Carrossel para [Cliente]” + mosaico 3×2.
  • 3 — Projeto 02: mesmo layout + novo cliente.
  • 4 — Projeto 03: mesmo layout + novo cliente.
  • 5 — Projeto 04: mesmo layout + novo cliente.
  • 6 — Projeto 05: mesmo layout + novo cliente.
  • 7 — Projeto 06: mesmo layout + novo cliente.
  • 8 — Bônus (opcional): “Antes/depois” (1 mini comparativo) ou “o que mudou no design”.
  • 9 — Micro-prova (opcional): 1 frase de resultado/feedback + miniaturas pequenas.
  • 10 — CTA: DM com palavra + “qual foi seu favorito?”

Elementos visuais (DNA do layout)

  • Formato: 1080×1350 (4:5).
  • Respiro: margens externas 7–10% (consistentes).
  • Topo fixo: 15–20% da altura (título do projeto).
  • Grade: 3×2 ocupando 70–75% da altura; gap 2–4%.
  • Separação: borda 1px ou sombra sutil nas miniaturas.
  • Acabamento: textura/grain (10–25%) para unificar.
  • Branding: @ no rodapé com baixa opacidade (sempre no mesmo lugar).

Tipografia (3 famílias): serif editorial (autoridade) + sans neutra (funcional) + script/brush (ênfase emocional).

Copy e ganchos

Ganchos de capa (escolha 1):

  • “Meus carrosséis favoritos de 2026 (em 10 slides)”
  • “Os 7 carrosséis que mais geraram pedido de orçamento”
  • “Top 8 layouts que eu repetiria hoje sem mudar nada”
  • “O meu estilo em 10 projetos (pra você ‘ver’ em 30s)”

Legenda sugerida (pronta para postar):

Meus carrosséis favoritos do ano — pra você ver meu estilo em poucos slides. Quer que eu crie um carrossel nesse nível pro seu perfil/marca? Me chama no direct com a palavra: CARROSSEL. Agora me diz: qual projeto foi seu favorito? (comenta o número) E salva esse post pra usar como referência ✦

CTA (modelo de 2 passos)

  • CTA principal (conversão): “Me chama no direct com a palavra CARROSSEL.”
  • CTA secundário (alcance): “Salva pra usar de referência.”
  • Engajamento (fácil): “Comenta o número do seu favorito.”
  • Opção ‘link completo’: “Comenta QUERO que eu te mando o carrossel inteiro.”

Checklist de criação (produção sem erro)

Conteúdo

  • [ ] 6–10 projetos selecionados (variedade real)
  • [ ] 6 miniaturas por projeto (coerentes e legíveis)
  • [ ] Títulos padronizados: “Carrossel para ___”
  • [ ] CTA final com palavra-chave de DM

Design

  • [ ] 1080×1350, margens 7–10%
  • [ ] Header fixo (altura consistente)
  • [ ] Grade 3×2 com gaps iguais
  • [ ] Borda/sombra leve nas miniaturas
  • [ ] Textura/grain constante (10–25%)
  • [ ] @ no rodapé (mesma posição e opacidade)

Qualidade (antes de publicar)

  • [ ] Teste de leitura em miniatura (tela do celular afastada).
  • [ ] Contraste suficiente (título e labels).
  • [ ] Consistência: alinhamentos idênticos entre slides.

Variações (mesmo formato, novos temas)

  • Educa + prova: “O que eu aprendi criando 30 carrosséis” (cada mosaico vira um ‘aprendizado + exemplo’).
  • Segmentado: “Meus melhores carrosséis para [nicho]” (ex.: saúde, moda, infoprodutor).
  • Before/After: cada slide mostra 1 mini “antes” + mosaico “depois”.
  • Metodologia: “Meu processo em 6 projetos” (briefing → wireframe → layout final).
  • Oferta suave: “Abrindo 3 vagas do mês: carrossel + identidade visual” (último slide com datas).
