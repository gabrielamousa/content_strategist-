Pare de usar #000000: 5 pretos melhores para design

Carrossel replicável para ensinar uma “troca inteligente”: explique por que o preto puro costuma piorar a leitura/sofisticação e entregue uma lista salvável com alternativas (HEX + quando usar). Estilo recomendado: editorial com textura/ruído e marcações manuscritas.
Objetivo: salvar + compartilhar
Entrega: 5 alternativas (1 por slide)
Tempo de produção: 45–90 min
Ver slides sugeridos Ir para checklist

Resumo

uso: conteúdo utilitário “pra salvar”
Ideia central: “Pare de usar #000000” (hook contrarian) + “aqui vão 5 substituições melhores”.
Por que funciona: padrão previsível, micro-recompensa por slide, comparativo (evitar vs substituir) e CTA de salvar logo no começo.
Como adaptar para outros temas: mantenha a estrutura (erro comum → por que dói → lista de 5 melhorias → fechamento com princípio) e troque apenas: “o que evitar” + “alternativas” + “critério de uso”.
Regra de ouro: cada slide da lista entrega 1 unidade completa: nome memorável + dado (HEX/atalho/passo) + quando usar.

Estrutura (macro)

8 slides • 1080×1350 (4:5)
  • Slide 1: Capa (hook + promessa + “salvável”).
  • Slide 2: Contexto (dor + consequência) + micro-CTA “salva”.
  • Slides 3–7: Lista (1 alternativa por slide) com padrão visual consistente.
  • Slide 8: Fechamento (princípio + pergunta + CTA final).

Roteiro (texto reproduzível)

cole no Canva/Figma/PS
SLIDE 1 — CAPA [Palavra em destaque (script)]: Pare [Título bold]: de usar #000000 [Sub]: 5 pretos melhores para design (com HEX) [Selo]: “pra você salvar” + “feito no Photoshop/Canva/Figma” (opcional) SLIDE 2 — CONTEXTO (DOR + PROMESSA) Headline: Preto puro (#000000) quase sempre deixa o layout mais duro. Apoio (1–2 linhas): Em fundos claros ele cria contraste agressivo; em alguns casos “quirta” a elegância e pesa a leitura. Micro-CTA: Salva esse post — você vai usar sempre. SLIDES 3–7 — LISTA (1 por slide, mesmo layout) Template do slide: Topo pequeno: Evite: #000000 (com X/riscado) Card principal: • Nome da alternativa: [NOME MEMORÁVEL] • HEX: [#XXXXXX] • Quando usar: [1 frase curta e prática] Rodapé: mini dica/variação (ex.: “use em títulos”, “use em textos longos”, “para UI dark”) Sugestões de alternativas (ajuste conforme seu estilo): 3) “Preto carvão” — #111111 — bom para texto sem ficar ‘chapado’. 4) “Preto grafite” — #1A1A1A — ótimo em UI e fundos escuros com respiro. 5) “Preto quente” — #1B1612 — funciona com paletas terrosas/editoriais. 6) “Preto frio” — #101826 — combina com tech/azulados, mantém sofisticação. 7) “Preto esverdeado” — #0F1A14 — bom para marcas naturais/premium. SLIDE 8 — FECHAMENTO Headline: Pequenas escolhas de cor mudam a percepção do design. Apoio: Trocar o preto puro por um ‘quase preto’ dá mais profundidade e refinamento. Pergunta: Qual desses você vai testar primeiro? CTA: Salva + envia pra um designer amigo. Quer parte 2? Comenta “PRETO”.

Elementos visuais (direção de arte)

editorial + textura
  • Fundo: foto/texture desaturada + papel/scan (20–45%) + ruído (6–12%).
  • Paleta: neutros quentes + 1 acento fixo (mostarda/dourado) para setas/círculos.
  • Cards: retângulos com cantos suaves (40–60px) e padding generoso (48–64px).
  • Marcações: sublinhado irregular, setas curvas, círculos tipo anotação, “X” riscado no #000000.
  • Ritmo: slides 3–7 iguais (catálogo) e um único slide com variação (ex.: slide 7) para quebrar monotonia.
Safe area: mantenha margens de 80–120px para o texto respirar e não encostar na borda.

Copy e ganchos (biblioteca rápida)

hooks curtos
Ganchos de capa (escolha 1):
  • “Pare de usar #000000.”
  • “Seu design tá ‘duro’? Pode ser o preto puro.”
  • “5 pretos melhores que #000000 (com HEX).”
Frases de contexto (slide 2):
  • “Preto puro dá contraste agressivo e ‘achata’ a profundidade.”
  • “Quase pretos deixam o layout mais elegante sem perder legibilidade.”
Micro-CTAs ao longo da lista:
  • “Salva pra consultar quando for escolher paleta.”
  • “Se você usa UI/brand, isso aqui resolve rápido.”

CTA (chamada para ação)

primário: salvar
  • CTA principal: “Salva esse post” (capa e slide 2).
  • CTA secundário: “Comenta ‘PRETO’ pra parte 2” (slide 8).
  • Compartilhamento: “Envia pra um amigo designer” (slide 8).
  • Engajamento leve: pergunta simples: “Qual você vai testar primeiro?”

Checklist de criação (do briefing ao publish)

produção
Conteúdo
  • ( ) Definir “o que evitar” (ex.: #000000) e o porquê em 1 frase.
  • ( ) Listar 5 alternativas com: nome memorável, HEX, quando usar.
  • ( ) Garantir que cada slide 3–7 entregue uma unidade completa.
  • ( ) Inserir “pra você salvar” até o slide 2.
  • ( ) Fechar com princípio + pergunta + CTA (salvar/compartilhar/comentar).
Design
  • ( ) Documento 1080×1350; margens 80–120px.
  • ( ) 2 famílias tipográficas (Sans + Script) no máximo.
  • ( ) Textura papel (20–45%) + ruído (6–12%) consistentes.
  • ( ) 1 cor de acento fixa para setas/círculos.
  • ( ) Assinatura @/quadro no mesmo lugar em todos os slides.
Antes de publicar
  • ( ) Slide 1 entendível em 1 segundo (leitura rápida no mobile).
  • ( ) Contraste ok e texto não encosta nas bordas.
  • ( ) Último slide tem CTA claro.

Slides sugeridos (1..8)

estrutura pronta
  1. Capa: “Pare de usar #000000” + “5 pretos melhores (HEX)” + selo “pra salvar”.
  2. Contexto: 1 frase de dor + 1 frase de promessa + “salva”.
  3. Alternativa 1: nome + HEX + quando usar (texto).
  4. Alternativa 2: nome + HEX + quando usar (UI/tech).
  5. Alternativa 3: nome + HEX + quando usar (editorial/print).
  6. Alternativa 4: nome + HEX + quando usar (paletas quentes).
  7. Alternativa 5: variação visual do layout (mesmo sistema, estética diferente).
  8. Fechamento: princípio + pergunta + CTA (salvar/compartilhar/comentar).

Variações (para repetir a ideia sem cansar)

séries
  • Versão “UI”: “Pare de usar preto puro no dark mode” + 5 quase-pretos + exemplo de componente (card/navbar).
  • Versão “tipografia”: “Pare de usar fonte X em tudo” + 5 substituições por objetivo (editorial, tech, branding, UI, títulos).
  • Versão “paleta”: “Pare de usar branco puro #FFFFFF” + 5 off-whites + quando usar (papel, web, moda, minimal).
  • Versão “impressão”: “Pare de usar 100K” + alternativas (rich black) + cuidados de registro.
  • Versão “erro comum”: “Pare de centralizar tudo” + 5 alinhamentos mais fortes + exemplos.
Dica de consistência: transforme isso em quadro fixo (ex.: “Trocas Inteligentes”) e mantenha sempre o mesmo padrão de slides 3–7.
