Carrossel de portfólio: template de cases “favoritos do ano”

Roteiro reproduzível para apresentar 4–8 projetos com consistência visual, prova de habilidade e fechamento por DM (sem parecer “venda”).

Ver post original

Resumo

  • Ideia central: “Meus [X] favoritos de [ANO]” + sequência de cases com o mesmo layout.
  • Por que funciona: padrão repetível (fácil de escanear) + variedade (curiosidade) + clareza do que você entrega (reduz objeções).
  • Objetivo oculto: portfólio + autoridade + convite para orçamento via Direct/link.
Regra de ouro: 1 template, N cases. A consistência é a “moldura” que valoriza seus trabalhos.

Estrutura

  • Canvas: 1080×1350 (4:5), margens 80–120px, corners 24–40px.
  • Sequência: Capa → (4 a 8) slides de case → slide final com CTA.
  • Template do case: “Para:” + bloco “O que é entregue” (2 colunas) + grid 2×2 + hero vertical + 4 tags.
  • Ritmo: 1 case por slide (evita poluição) e mantém previsibilidade.

Slides sugeridos (1..10)

Slide 1 — Capa (hook + curadoria)
Título: “Meus [X] favoritos de [ANO]” + microtexto “Portfólio de [categoria]”. Imagem lifestyle/bastidor + sua assinatura.
Slide 2 — Case 01
Para: [Cliente/Projeto] + entregáveis + grid + 4 tags.
Slide 3 — Case 02
Mesmo layout; mude paleta e imagens para reforçar variedade.
Slide 4 — Case 03
Repetição = retenção. Evite alterar hierarquia do template.
Slide 5 — Case 04
Se o projeto for forte, use hero vertical maior (ainda consistente).
Slide 6 — Case 05 (opcional)
Só inclua se sustentar qualidade e variedade.
Slide 7 — Case 06 (opcional)
Mantenha o “Para:” no mesmo lugar (âncora visual).
Slide 8 — Case 07 (opcional)
Feche a série no pico (seu melhor case por último funciona bem).
Slide 9 — Case 08 (opcional)
Se tiver 8+, enxugue texto para não cansar.
Slide 10 — Fechamento (CTA editorial)
Pergunta + instrução simples (palavra-chave no direct) + incentivo de salvar/compartilhar.

Roteiro (reproduzível)

CARROSSEL — “MEUS [X] FAVORITOS DE [ANO]” SLIDE 1 — CAPA Título (3 linhas): • Meus [X] • favoritos • de [ANO] Microtexto: PORTFÓLIO DE [CATEGORIA] Rodapé: @SEUUSER SLIDES 2..(N-1) — CASE (repita o mesmo template) Cabeçalho: Para: [CLIENTE/PROJETO] Bloco 2 colunas: • Esquerda (título): O QUE É ENTREGUE EM CADA PROJETO: • Direita (texto curto, 1–2 linhas): Paleta de cores, combinações tipográficas, moodboard do universo visual, assinatura gráfica e [N] templates editáveis no [FERRAMENTA]. Grid central: • 4 miniaturas (2×2) + 1 hero vertical (mockup de celular/cartaz/aplicação) Rodapé (tags em cápsulas): [ADJETIVO 1] [ADJETIVO 2] [ADJETIVO 3] [ADJETIVO 4] Assinatura: @SEUUSER SLIDE N — CTA (fechamento) Título: Quer um alinhamento visual assim? Linhas/ações: • → Envie “ALINHAMENTO” no direct • → Ou peça orçamento no link da bio Extra: Salve este carrossel para se inspirar + compartilhe com alguém que precisa.
Dica: deixe o texto do bloco “O que é entregue…” praticamente igual em todos os cases (muda apenas o número de templates ou a ferramenta). Isso comunica método.

Elementos visuais

  • Direção de arte: editorial minimalista, bastante respiro, composição centralizada.
  • Hierarquia tipográfica (60/30/10): 60% Sans (info), 30% Serif (autoridade), 10% Script (picos de atenção na capa).
  • Componentes fixos: “Para:”, bloco 2 colunas, grid 2×2 + hero, tags cápsula, @ discreto.
  • Imagens: mockups e aplicações reais; miniaturas com cantos arredondados; sombra mínima.
margens 80–120 grid 2×2 + hero tags em cápsulas padrão repetível

Copy e ganchos

  • Hook de capa (curadoria + novidade): “Meus [X] favoritos de [ANO]”.
  • Hook por case (prova social): “Para: [Cliente/Projeto]”.
  • Hook de valor (clareza): “O que é entregue em cada projeto: …”.
  • Hook de comparação (rápido): 4 adjetivos resumindo posicionamento.
Banco rápido de “4 adjetivos” (copiar e escolher)
Técnico/negócio: estratégica, clara, objetiva, premium. • Emocional: acolhedora, leve, confiante. • Estética: clean, sofisticada, moderna, editorial, orgânica, vibrante. • Tom: jovem, autêntica, criativa, minimalista, ousada.

CTA

  • Principal (DM): “Envie ‘ALINHAMENTO’ no direct”.
  • Alternativa (link): “Peça orçamento no link da bio”.
  • Retenção: “Salve para usar como referência” + “Compartilhe com alguém que precisa”.
Micro-roteiro de DM (opcional): “Me diz seu nicho + o que você vende + 2 referências de estética. Eu te respondo com prazos e próximo passo.”

Checklist de criação

  • Antes: definir tema ([X] favoritos), ano, e 4–8 cases.
  • Por case: nome do projeto + 4–6 imagens + 4 adjetivos + texto padrão de entregáveis.
  • Design: montar 1 template (master) e duplicar para todos os slides.
  • Consistência: mesma posição para “Para:”, bloco 2 colunas e tags.
  • Finalização: revisar legibilidade (tamanho de fonte, contraste), padronizar bordas, exportar em 1080×1350.
  • Publicação: legenda curta + CTA, fixar nos destaques/portfólio, responder DMs com perguntas padrão.

Variações (mesmo template, novos temas)

Variação 1 — “Rebrands favoritos”
Troque o bloco “entregáveis” por “o que mudou” (Antes/Depois em 1 linha).
Variação 2 — “Capas favoritas”
Grid vira 6 miniaturas + 1 hero; tags viram “gênero / clima / cor / referência”.
Variação 3 — “Identidades por nicho”
Organize cases por categoria (ex.: saúde, beleza, tech) e use cor de destaque por nicho.
Variação 4 — “Processo”
Em cada case, troque 2 miniaturas por: briefing, moodboard, tipografia.
