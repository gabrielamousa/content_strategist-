Carrossel “Vícios de design”: roteiro replicável em 7 slides

Objetivo: lista curta + demonstração visual + participação (comentários/salvos).

Resumo

Ideia central: “Confissão + lista” (5 itens) com alternância de densidade visual (colagem vs tipográfico) para manter ritmo e retenção.
Entrega: cada item tem nome curto + exemplo visual + (opcional) 1 linha do porquê.
Fecho: CTA de escolha (“qual é mais a minha cara?”) + convite para salvar.
Hook pessoal ("eu não largo")Lista (cérebro quer completar)Slides curtos (1 ideia)Prova visual (print/mock)Pergunta final (comentários)

Estrutura

Esqueleto reproduzível (7 slides): Capa → 5 itens → Encerramento. Mantém consistência visual com 2–3 elementos fixos (cor assinatura + grain + forma/ícone + @).

  • S1
    Capa (Hook + promessa)
    Título forte + subtítulo pessoal/curioso + visual “hero” (foto recortada/objeto) + 1 elemento de processo (print UI opcional).
  • S2–S6
    Itens em capítulo (1 ideia por slide)
    Nome curto do “vício” + exemplo aplicado. Opcional: 1 linha de benefício. Alterne: 1 slide “cheio” → 1 slide mais tipográfico/minimal.
  • S7
    Encerramento (interação + ação)
    Pergunta de preferência + micro CTA (comentar/salvar/marcar). Mantenha a assinatura (@) e o elemento recorrente.

ROTEIRO — Carrossel “Vícios/Manias/Recursos” (7 slides)

Slides sugeridos (1..7)

Use 1080×1350 (4:5). Regra de ouro: 1 ideia por slide, texto curto e exemplo grande.

Slide 1
Capa / Confissão
Hook + subtítulo pessoal + hero recortado. Elemento fixo: grain + forma/estrela + @.
Slide 2
Item 1 (cheio)
Exemplo forte + nome curto do vício + 1 linha de benefício.
Slide 3
Item 2 (tipográfico)
Big type / caixas. Menos elementos para dar respiro.
Slide 4
Item 3 (cheio)
Colagem com recorte irregular + print UI opcional.
Slide 5
Item 4 (tipográfico)
Palavra gigante no fundo + card/label com a dica.
Slide 6
Item 5 (cheio)
O mais “uau” (finaliza alto) + micro prova (antes/depois).
Slide 7
Fecho (interação)
Pergunta de escolha + CTA salvar + convite marcar alguém.

Elementos visuais

  • Identidade (repetições que criam “série”)
    Escolha 2–3 fixos: cor assinatura + grain/ruído + forma (estrela/recorte) + assinatura (@).
  • Composição em camadas (colagem)
    Fundo texturizado → forma grande → foto/print recortado → título grande → detalhe pequeno.
  • Alternância de densidade
    Intercale slides “lotados” (colagem) com slides minimal (big type) para reduzir fadiga e aumentar retenção.
  • Prova visual rápida
    Use print do software, mock ou antes/depois (mesmo que mini) para ancorar credibilidade.

Copy e ganchos

Fórmula: confissão + lista + prova + pergunta. Abaixo, opções substituíveis com o mesmo impacto.

  • Ganchos de capa (escolha 1)
    Opções
    • “Vícios de [TEMA] que eu não largo”
    • “5 manias de [TEMA] que deixam tudo mais profissional”
    • “Coisas que eu sempre uso em [TIPO DE PEÇA]”
    • “Meu estilo em 5 recursos (sem culpa)”
  • Modelo de item (1 linha)
    • “Uso quando quero [benefício].”
    • “Meu atalho para [resultado].”
    • “Funciona bem em [contexto].”
    • “Se você só testar 1 hoje: [nome].”
  • Fecho que puxa comentário
    • “Qual é mais a minha cara: 1, 2, 3, 4 ou 5?”
    • “Qual você mais usa?”
    • “Qual você quer que eu faça um tutorial?”

CTA

  • CTA principal (comentários)
    “Comenta o número (1–5) que você mais usa / que você mais gostou.”
  • CTA secundário (salvamento)
    “Salva pra usar como checklist no próximo job.”
  • CTA de alcance (marca/compartilha)
    “Marca um(a) [perfil] que vai amar [tema].”
  • CTA de produto (opcional)
    “Quer o template? Comenta ‘TEMPLATE’ (ou manda DM com ‘TEMPLATE’).”

Checklist

  • Planejamento
    [ ] Tema definido em 5 itens (nomes curtos).
    [ ] Para cada item: exemplo visual + (opcional) benefício em 1 frase.
    [ ] Ordem dos itens do “bom” → “melhor” (termina alto).
  • Design (consistência)
    [ ] 2–3 elementos recorrentes (cor + grain + forma + @).
    [ ] Alternância: 1 slide minimal a cada 1–2 slides cheios.
    [ ] Máx. 2 fontes; H1 gigante; texto curto.
  • Legibilidade mobile
    [ ] Teste em 25% de zoom (tudo legível).
    [ ] Contraste alto (texto vs fundo).
    [ ] Margens seguras (evitar bordas muito próximas).
  • Publicação
    [ ] Último slide com pergunta + ação (comentar/salvar).
    [ ] Legenda com lista repetida (facilita skim).
    [ ] Primeiro comentário com “template/recursos” (opcional).

Variações

Trocas de tema mantendo o mesmo efeito (confissão + lista + prova + escolha):

  • Variação A — “Vícios” vira “Sinais de profissional”
    Título: “5 sinais de que seu [tema] está mais profissional”. Itens viram critérios (ex.: hierarquia, contraste, espaçamento, consistência, alinhamento) + exemplo.
  • Variação B — “Vícios” vira “Erros comuns (e como eu resolvo)”
    Cada slide: erro → correção visual (antes/depois). Fecho: “Qual erro você mais comete?”
  • Variação C — “Minha assinatura” (autoridade)
    Título: “As 5 coisas que sempre aparecem no meu trabalho”. Itens: decisões recorrentes + prova (print de projetos).
  • Variação D — Para outro nicho (substituições rápidas)
    Substitua:
    • “print do software” → “print da planilha / ferramenta / bastidor”
    • “recorte/foto” → “produto/cliente/cena”
    • “vícios de design” → “hábitos de [profissão]”
    Mantém o motor: lista + prova + pergunta.
Identidade: fundo escuro, cards com borda suave, destaques em degradê (verde/azul/laranja), layout limpo high-tech.Dica: transforme em série semanal (Dia X) para reconhecimento.
