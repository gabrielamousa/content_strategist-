7 fontes para títulos que puxam atenção (template de carrossel)

Roteiro reproduzível de carrossel “curadoria + utilidade”, com padrão visual consistente (tipo como imagem), alta retenção (seta fixa + repetição) e alto potencial de salvamento (lista simples e aplicável).

Resumo

O que você vai produzir

Objetivo: entregar uma curadoria rápida (referência) com demonstrações visuais impactantes e informação mínima (nome do item), levando a salvar e seguir.

Fórmula
Promessa (capa) → Coleção (exemplos) → Ação (CTA)
Tamanho
1080×1350 (4:5), fundo escuro
Retenção
Seta fixa + repetição de estrutura + variação estética
Regra de ouro: 1 slide = 1 item da lista. Texto gigante, pouca palavra, alto contraste.

Estrutura

Sequência fixa (reproduzível)

Quantidade sugerida: 8 slides (1 capa + 6 itens + 1 CTA). Ajustável para 7, 9 ou 10 mantendo a mesma lógica.

  • Slide 1 (Capa): promessa + indicação de lista + micro-CTA leve (ex.: “salva”).
  • Slides 2–7 (Valor): demonstração visual do item + label com nome + UI fixa (logo e seta).
  • Slide 8 (CTA): 1 frase direta + 1 ação principal (seguir/salvar/comentar).

Componentes fixos (UI do carrossel):

  • Assinatura/Marca no canto inferior esquerdo (consistência).
  • Seta “→” no canto inferior direito (sinal de continuidade).
  • Label inferior: “ITEM.” + “NOME” (+ specs opcionais).
  • Fundo escuro + destaque em 1 cor (não competir com o texto).

Elementos visuais

Guia de identidade

Layout (Figma/Canva):

  • Documento 1080×1350, margem segura 80–120px.
  • Texto herói ocupa 70–90% do slide, quase encostando nas bordas.
  • Alinhamento principal à esquerda (permite variações por ritmo).
  • Repetição de UI: marca (esq. baixo) + seta (dir. baixo) em todos os slides.
Alta retenção: o usuário entende em 2 segundos que “cada slide = novo item”.

Paleta sugerida (adaptável):

  • BG: #000000 ou #070A0F
  • Texto principal: off-white (#E7EFE3 / #EAF1FF)
  • Acento único: dourado/areia (ou substitua por verde/azul/laranja)

Detalhes que elevam:

  • Microtexto em caps com tracking maior (label).
  • Um único elemento gráfico discreto (ícone/forma) só na capa (ou 1 em 3 slides).
  • Não usar sombras pesadas no texto: contraste deve vir do tamanho e cor.

Template de substituição (mesmo impacto em qualquer tema):

  • Troque FONT. por TOOL., COLOR., PROMPT., IDEIA., LAYOUT., ERRO., etc.
  • Troque o “hero text” por um exemplo típico do tema (curto e visual).
  • Mantenha a lógica “nome + microinfo” para aumentar salvamentos.

CTA

Escolha 1 objetivo
  • Para salvar (o melhor para curadoria): “Salva esse post pra consultar quando for criar [resultado].”
  • Para seguir: “Quer mais listas assim? Segue o perfil — eu posto curadorias toda semana.”
  • Para comentar (engajamento): “Comenta ‘LISTA’ que eu te mando os links/nomes em um bloco.”
  • Para clicar (produto/material): “Pack completo + extras: confere o link na bio.”
Dica: coloque um micro-CTA “SALVA” discreto já no Slide 1 para aumentar saves.

Checklist

Criação → publicação

Checklist de criação (design):

  • Arquivo 1080×1350; margens seguras definidas.
  • BG escuro + 1 acento (não mais que 10–15% da área).
  • Hero text gigante e legível em tela pequena.
  • Label inferior com padrão fixo: “ITEM. NOME” (+ specs opcionais).
  • Marca e seta na mesma posição em todos os slides.
  • Revisar consistência: tamanhos, alinhamento, espaçamentos.

Checklist de conteúdo (alto desempenho):

  • Capa com promessa clara + número (quando possível).
  • Um item por slide (sem tentar ensinar demais).
  • Informação “salvável”: nomes, referências, parâmetros, exemplos.
  • Último slide com 1 CTA principal (e 1 secundário no máximo).
  • Legenda curta com: gancho + pra quem é + CTA.
  • Publicar em horário de pico do seu perfil e responder comentários na 1ª hora.

Variações

Mesma estrutura, outros temas
  • Ferramentas: “7 TOOLS para [tarefa]” + label “TOOL. Nome” + microinfo (preço/uso).
  • Cores: “7 PALETAS para [estilo]” + hero com nome do estilo + label com hex codes.
  • Prompts (IA): “7 PROMPTS para [resultado]” + hero com 2–4 palavras do efeito + label com prompt resumido.
  • Layouts: “7 COMPOSIÇÕES para [peça]” + hero com mini regra (ex.: “Grid 3×3”) + label com espaçamentos.
  • Erros comuns: “7 ERROS em [tema]” (vira pilar ‘Erro comum’) mantendo seta e repetição.
Substituições que mantêm o impacto: número + “para” + objetivo (capa) / exemplo visual curto (slides) / nome do item (label) / CTA de salvar (final).
Dica operacional: salve este HTML como um “template do dia” e duplique para novos temas trocando apenas: promessa da capa, lista de itens e CTA.
