Hook minimalista: “Toda noite” + causa + 3 passos

Use um início incompleto (1–4 palavras) para abrir loop de curiosidade, espelhar uma sensação recorrente e entregar uma solução curta com 2–3 ações. Ideal para dor, hábito, rotina e consistência.

Abrir post original
Dica: troque “Toda noite” por “Todo dia/segunda/manhã” mantendo a mesma estrutura.

Resumo

Objetivo: retenção + comentários

Fórmula: Hook incompleto → identificação → quebra de crença → causa real (1) → passos práticos (2–3) → CTA com palavra-chave + salvar.

Quando usar

Quando você quer transformar um comportamento repetido (dor, fome, procrastinação, “amanhã eu treino”) em uma explicação simples e acionável em menos de 35s.

Estrutura

20–35s (até 45s)
  1. 0–2s — Hook: 1–4 palavras, incompleto.
  2. 2–6s — Espelho: descreva a sensação/situação exata.
  3. 6–10s — Tensão: “você acha que é X… mas é Y”.
  4. 10–18s — Causa real: 1 explicação simples (não dê 5 motivos).
  5. 18–30s — Passo a passo: 2–3 ações curtas em lista.
  6. 30–35s — Fechamento: CTA (comentar palavra-chave) + “salvar”.

Regra de ouro: uma ideia por tela e mudança a cada 1–2 segundos (corte/zoom/texto novo).

Roteiro (modelo reproduzível)

Pronto para copiar
[CENA 1 • 0–2s • TEXTO GRANDE NA TELA]
"TODA NOITE."
(olhar para câmera / deitar na cama / mão na lombar)

[VOZ • 2–6s]
"Você deita e a (FOME / DOR / MENTE ACELERADA) aparece."

[VOZ • 6–10s • QUEBRA DE CRENÇA]
"E você acha que é falta de disciplina… mas não é."

[VOZ • 10–18s • CAUSA REAL (1 FRASE)]
"O problema é (ESTRUTURA/AMBIENTE/ROTINA): (1 causa simples e clara)."

[VOZ • 18–30s • PASSO A PASSO (2–3 ITENS)]
"Faz isso hoje:"
"1) (ação #1 em 5–8 palavras)"
"2) (ação #2 em 5–8 palavras)"
"3) (ação #3 opcional, em 5–8 palavras)"

[FECHAMENTO • 30–35s]
"Comenta 'NOITE' que eu te mando um protocolo." 
"E salva pra fazer hoje."

Campos para preencher

  • Padrão: toda noite / todo dia / toda manhã
  • Sensação: fome, dor, ansiedade, culpa
  • Causa real: 1 explicação simples
  • Ações: 2–3 passos curtos
  • Palavra-chave: “NOITE” (ou tema)

Exemplos prontos (Educador Físico)

  • Dor lombar: “2h sentado + zero mobilidade” → 2 exercícios
  • Fome noturna: “pouca proteína de dia” → jantar estruturado
  • Consistência: “treino grande demais” → plano mínimo viável
  • Recuperação: “tela + cafeína tarde” → rotina de desligamento

Elementos visuais

Edição high-retention
  • Primeiro frame limpo: fundo escuro + texto grande “TODA NOITE.”
  • Jump cuts: a cada frase (1–2s) para manter ritmo.
  • Zoom leve: na quebra de crença (“não é disciplina”).
  • B-roll (2–3 takes): cama/relógio/noite; preparar refeição; alongamento/respiração.
  • Legendas queimadas: sempre (voz + texto reforçando).
  • Texto na tela por telas: 1 ideia por tela (sem parágrafos).

Texto na tela (sequência)

1) TODA NOITE.
2) A (fome/dor) volta.
3) Você acha que é disciplina…
4) Mas é ESTRUTURA.
5) Faz isso hoje:
6) 1) ____
7) 2) ____
8) 3) ____
9) Salva pra fazer à noite.

Copy e ganchos

Substituições com mesmo impacto

Hooks minimalistas (1–4 palavras)

  • “Toda noite.”
  • “Todo dia.”
  • “Toda segunda.”
  • “De novo.”
  • “Sempre igual.”

Linhas de identificação

  • “Você deita e a lombar reclama.”
  • “Você promete que amanhã treina.”
  • “A fome aparece do nada.”
  • “A mente não desliga.”

Quebra de crença (trocas)

  • “Não é disciplina. É estrutura.”
  • “Não é preguiça. É planejamento.”
  • “Não é falta de tempo. É prioridade mínima.”
  • “Não é idade. É hábitos.”

Promessa curta (choose one): “Entenda por que acontece e o que fazer em 30s.” • “1 ajuste hoje, menos (dor/fome) à noite.” • “Treino consistente sem motivação.”

CTA

1 principal + 1 secundário
  • Principal (comentário): “Comenta ‘NOITE’ que eu te mando um protocolo.”
  • Secundário (salvar): “Salva pra fazer hoje antes de deitar.”

Outras CTAs (mesmo efeito)

  • “Quer a versão para (objetivo)? Comenta ‘QUERO’.”
  • “Me manda ‘NOITE’ no direct.”
  • “Marca alguém que sofre com isso.”

Palavras-chave por tema

  • Dor: “LOMBAR” / “PESCOÇO”
  • Emagrecimento: “FOME”
  • Consistência: “ROTINA”
  • Corrida: “CANSAÇO”

Checklist

Criação + gravação + edição

Pré-roteiro

  • Escolha 1 padrão: toda noite / todo dia / toda manhã.
  • Defina 1 dor específica (sensação + contexto).
  • Defina 1 crença errada (“é disciplina/idade/tempo”).
  • Escolha 1 causa real (simples e explicável em 1 frase).
  • Liste 2–3 ações que cabem em 10–15 min no total.

Roteiro

  • Hook incompleto (1–4 palavras).
  • Espelho com sensação literal (“deita e…”).
  • Quebra de crença (“não é X, é Y”).
  • Passo a passo em lista (2–3 itens).
  • CTA com palavra-chave + salvar.

Gravação

  • Primeiro frame sem poluição visual; texto grande.
  • Áudio limpo (voz em primeiro plano).
  • Performance: pausas curtas após frases-chave.
  • B-roll (2–3 takes) que “mostram a noite”.

Edição

  • Cortes a cada 1–2s (sem monólogo longo).
  • Legenda sempre; destaque palavras (“DISCIPLINA”, “ESTRUTURA”).
  • Zoom leve na tensão/quebra de crença.
  • Última tela com CTA grande + palavra-chave.

Reprodutibilidade: mantenha o mesmo esqueleto e troque apenas (padrão + sensação + causa + 2–3 ações + palavra-chave).

Variações

Adaptações para outros temas

Trocas rápidas (mesmo impacto)

  • Tempo: “Toda noite” → “Toda manhã” (energia) / “Toda segunda” (começo).
  • Verbo: “volta” → “aparece” / “bate” / “trava” / “pesa”.
  • Crença: “disciplina” → “motivação” / “força de vontade” / “genética”.
  • Causa: “estrutura” → “ambiente” / “rotina” / “sinal do corpo”.

4 roteiros prontos (subtemas)

  • Fome noturna: causa = baixa proteína/fibra no dia; ação = jantar estruturado + água + rotina sem tela.
  • Dor no fim do dia: causa = rigidez + postura; ação = mobilidade 5 min + caminhada 10 min.
  • Sem consistência: causa = plano irreal; ação = treino mínimo (15 min) + agenda fixa.
  • Recuperação ruim: causa = estímulo alto à noite; ação = luz baixa + respiração + cafeína até horário X.

Se você quiser adaptar para outro nicho: mantenha “padrão repetitivo + quebra de crença + lista curta”. Troque apenas a sensação e o micro-protocolo.
