Reel hipnótico: a lista que revela seus sabotadores

Use uma sequência rítmica “É…, é…, é…” com itens curtíssimos para causar reconhecimento + curiosidade (retenção alta). O fechamento dá o insight; a explicação pode ir na legenda/comentário fixado.

Resumo

Objetivo do post
Identificação + insight rápido
Duração ideal
8–14s (loop natural)
Estratégia
Cadência + lista crescente
Entrega
Explica na legenda

Por que funciona: repetição (anáfora) cria padrão, baixa carga cognitiva e um “o que vem depois?” constante. Perfeito para temas de nutrição que são soma de micro-hábitos.

Metadados (calendário)

  • Título: Reel hipnótico: a lista que revela seus sabotadores
  • Instrução curta: Grave em cadência “É…, é…, é…” com 8–12 itens + 1 virada final + CTA único.
  • Pilar: Erro comum
  • Formato: Reels
Hook: “É…, é…, é…” Promessa: “Você vai se reconhecer” CTA: salva / comenta / envia

Estrutura (reproduzível)

  1. Abertura (0–1s): 1 item muito comum e direto (com palavra concreta).
  2. Escalada (1–10s): 8–12 itens de 1–4 palavras cada (com progressão: comum → específico → “dor”).
  3. Virada (10–14s): 1 frase que amarra o sentido (ou deixa aberto para loop).
  4. Legenda: 1–3 bullets com “o que ajustar primeiro” + CTA único.

Elementos visuais

  • Enquadramento: 9:16, plano médio/close alternando (jump cuts leves).
  • Fundo: limpo; luz frontal suave; contraste alto no texto.
  • Texto: sincronizado com a fala (1 palavra forte por item).
  • Loop: finalize com “aberto” (sem conclusão longa) para replay.
  • B-roll (0,3–0,6s): garrafa d’água, prato simples, mercado, marmita, café, relógio/agenda.
  • Som: voz em destaque + beat leve (ou trend discreta).
  • Ritmo: acelera levemente no meio; última frase mais marcada.
  • Tipografia: caixa alta para palavras-chave; sombra sutil no texto.

Copy e ganchos

Hooks (1ª linha do vídeo):

  • “É fome. É vontade. É ansiedade.”
  • “É ‘só um docinho’. É belisco. É ‘segunda eu volto’.”
  • “É inchaço. É retenção. É pouca água.”
  • “É cansaço. É café. É açúcar no fim do dia.”

Fechamentos (virada):

  • “E no fim… não é preguiça. É soma.”
  • “Você não falha: você repete um padrão.”
  • “Arruma 1 peça. O resto começa a encaixar.”
  • “Isso aqui é o que te trava.”

CTA

  • Para comentários: “Comenta ‘EU’ se você se reconheceu.”
  • Para salvamentos: “Salva pra rever quando bater a vontade.”
  • Para diagnóstico do público: “Qual item te pega mais: 1, 2 ou 3?”
  • Para gerar demanda: “Comenta ‘AJUSTE’ que eu faço a versão do seu objetivo.”

Regra: escolha 1 CTA por post. CTA demais derruba conversão.

Checklist de criação (operacional)

Pré-roteiro

  • Defina 1 tema com “cadeia de fatores” (inchaço, fome emocional, estagnação).
  • Liste 8–12 itens (1–4 palavras) que o público vive.
  • Ordene: comum → específico → consequência.
  • Escreva 1 frase final (virada) em até 8–10 palavras.
  • Decida: explicação no vídeo (mínima) ou só na legenda.

Gravação + edição

  • Grave 2 takes: um mais rápido e outro mais marcado.
  • Corte no beat (a cada item ou a cada 2 itens).
  • Texto na tela sincronizado (palavra-chave por item).
  • Insira 3–6 b-rolls curtos para variar sem quebrar a cadência.
  • Finalize com micro-suspensão (0,2–0,4s) para loop.

Publicação: legenda com 1–3 bullets “comece por aqui” + comentário fixado pedindo o tema da próxima versão.

Variações (3 roteiros rápidos)

V1 — Fome emocional “É cansaço. É irritação. É cabeça cheia. É ‘eu mereço’. É silêncio na cozinha. É belisco. É doce. É culpa. É recomeço. E no fim… é emoção pedindo cuidado.”
V2 — Inchaço “É comer rápido. É mastigar pouco. É refrigerante. É sal alto. É pouca água. É pouco potássio. É intestino lento. É ciclo hormonal. E no fim… é inchaço acumulando.”
V3 — Estagnação no emagrecimento “É fim de semana sem plano. É ‘só hoje’. É álcool. É pouco sono. É fome no meio da tarde. É jantar tarde. É proteína baixa. É passo baixo. E no fim… é déficit que não acontece.”

Dica UX: mantenha as palavras “visíveis” (concretas) e evite termos técnicos no vídeo. Se precisar, traduza na legenda.
