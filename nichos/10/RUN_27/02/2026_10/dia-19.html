Chega de dieta triste: 2 passos pra comida suculenta

Roteiro reproduzível (base + upgrade) com retenção por curiosidade, listas e payoff visual.

Post original: instagram.com/p/DTyXJKOAXFb/

Resumo

Este modelo performa bem porque começa com uma opinião forte (dieta não precisa ser triste), entrega valor prático imediato (2 partes + 3 usos) e ainda reduz fricção com meal prep (tempo de geladeira/freezer).

Gatilhos: quebra de crença
Retenção: “Parte 1 / Parte 2”
Valor: 3 usos do preparo
Compartilhável: CTA com humor
Objetivo: bater meta (proteína/fibra) com prazer
Método: técnica simples + molho/texture hack
Erro comum: corte magro + método que resseca
Payoff: textura (vapor, brilho, cremosidade)

Estrutura (sequência reproduzível)

  • 0–2s — Hook (dor + opinião) + close do resultado.
  • 2–6s — Micro-promessa + “vai virar repetida” + 3 usos (lista rápida).
  • 6–20s — Erro comum + 1 princípio técnico em 1 frase.
  • 20–45s — Execução em 2 partes: Base → Upgrade.
  • 45–55s — Opcional “segredo” (um detalhe que parece chef).
  • 55–60s — Armazenamento + CTA (salvar/compartilhar/comentar).

Regra de ouro

Uma ideia por bloco. Texto na tela sempre “marcando capítulo” e números grandes (2 partes / 3 usos / 5 dias / 60 dias).

Roteiro (template + exemplo pronto)

Substitua os campos entre colchetes para adaptar para qualquer tema, mantendo o mesmo impacto.

TÍTULO NA TELA: “Chega de [refeição/ingrediente] seco. Dieta não precisa ser triste.” [0–2s | CLOSE do resultado] VO: “Não dá pra normalizar comer [comida seca/sem graça] só porque você precisa bater [meta: proteína/fibra/caloria].” TEXTO: “Chega de [X] seco.” [2–6s | PROMESSA + 3 USOS] VO: “Fica aqui: isso vai virar a figurinha repetida do seu [almoço/lanche/jantar] da semana.” VO: “Serve pra: 1) [uso 1], 2) [uso 2], 3) [uso 3].” TEXTO: “3 usos (salva isso)” [6–20s | ERRO + PRINCÍPIO] VO: “O erro comum é [erro: escolher só corte magro + método errado / pouco molho / pouco tempero].” VO: “O que resseca é [princípio em 1 frase: perda de água por temperatura alta/tempo demais/falta de ‘meio’].” TEXTO: “Erro comum → Segredo” [20–32s | PARTE 1: BASE] TEXTO: “Parte 1: base” VO: “Você vai fazer assim: [passo base 1], depois [passo base 2].” CENAS: corte rápido + vapor + textura. [32–45s | PARTE 2: UPGRADE] TEXTO: “Parte 2: molho/upgrade” VO: “Agora o pulo do gato: [molho/tempero/textura].” VO: “É isso que deixa [cremoso/suculento/encorpado] e faz você bater [meta] com prazer.” CENAS: molho brilhando + colher puxando. [45–55s | OPCIONAL ‘SEGREDO’] TEXTO: “Opcional (fica absurdo)” VO: “Se quiser, coloca [hack: iogurte/limão/ervas/amido/mostarda/caldo] pra [efeito: encorpar/realçar sabor/segurar suculência].” [55–60s | MEAL PREP + CTA] TEXTO: “Geladeira: [X] dias | Freezer: [Y] dias” VO: “Dura [X] dias na geladeira e [Y] no freezer.” VO: “Salva pra fazer na semana e compartilha com quem vai cozinhar pra você… nem que seja você mesmo.” LEGENDAS (prontas): • “Dieta não precisa ser triste: 2 passos (base + upgrade) pra [resultado].” • “Comenta ‘QUERO’ que eu mando 5 variações de [proteína/lanche/marmita].” • “Salva pra usar no meal prep.”
Dica: grave 2 trechos olhando pra câmera (Hook e CTA). O resto pode ser só B-roll + texto.

Elementos visuais

Cena obrigatória (payoff)

  • Close do resultado antes do passo a passo (brilho, vapor, cremosidade).
  • Textura em evidência: desfiar/partir/puxar com colher.
  • Números grandes na tela: 2 partes, 3 usos, X dias/Y dias.

Edição (padrão de retenção)

  • Jump cuts a cada 1–2s durante explicações.
  • Capítulos na tela: “Erro”, “Segredo”, “Parte 1”, “Parte 2”, “Opcional”.
  • ASMR leve (mexer, cortar, fervura) + voiceover.

Copy e ganchos (substituíveis)

Hooks (escolha 1)

  • “Não dá pra normalizar [X sem graça] só porque você precisa [meta].”
  • “Isso vai virar a figurinha repetida do seu [momento do dia] essa semana.”
  • “É um erro achar que pra ficar gostoso precisa [vilão: óleo/queijo/açúcar].”

Frases de autoridade (sem virar aula)

  • “Eu já passei por isso: eu também achava que dieta era comida triste.”
  • “O ponto não é ‘receita fit’, é técnica pra não ressecar.”
  • “Pequenos ajustes = mais consistência (e mais resultado).”

CTA (modelos prontos)

  • Salvar: “Salva pra testar na próxima marmita.”
  • Compartilhar: “Compartilha com quem cozinha pra você… nem que seja você.”
  • Comentário (DM/engajamento): “Comenta ‘QUERO’ que eu mando [número] variações.”
  • Pesquisa: “Você prefere versão com [A] ou com [B]?”

Checklist de criação (produção rápida)

Pré-roteiro

  • [ ] Escolher 1 dor (ex.: “marmita seca”, “lanche sem graça”, “não bate proteína”).
  • [ ] Definir 1 promessa (sabor / saciedade / praticidade).
  • [ ] Listar 3 usos do preparo (marmita, sanduíche, ensopado etc.).
  • [ ] Definir 1 crença pra quebrar (ex.: “dieta precisa ser sem graça”).
  • [ ] Escrever 1 princípio técnico em 1 frase (“o que resseca é…”).
  • [ ] Montar em 2 partes: base + upgrade.

Gravação + edição

  • [ ] Abrir com close do resultado (2s).
  • [ ] Gravar B-roll do payoff: vapor, brilho, textura “puxando”.
  • [ ] 2–3 trechos falando pra câmera: Hook, Erro/Segredo, CTA.
  • [ ] Textos na tela em capítulos + números grandes.
  • [ ] Cortes a cada 1–2s nas falas.
  • [ ] Legenda: promessa + variações + armazenamento + CTA.

Variações (adaptações com o mesmo impacto)

Use a mesma estrutura “dor → promessa → 3 usos → erro → segredo → 2 partes → storage → CTA”, trocando apenas os componentes abaixo.

Trocas rápidas (plug-and-play)

  • [Meta]: proteína / fibra / saciedade / pré-treino / emagrecimento sustentável.
  • [Vilão]: “precisa de muito óleo” / “precisa de queijo” / “precisa de açúcar”.
  • [Upgrade]: molho proteico (iogurte/ricota), molho de tomate encorpado, caldo + amido, ervas + cítrico.
  • [Opcional]: limão, mostarda, páprica defumada, ervas frescas, iogurte, amido (pouco) pra encorpar.

3 temas prontos (mesma espinha)

  • Lanche proteico: “Chega de lanche seco — 2 passos pra ficar cremoso e saciante.”
  • Jantar com legumes: “Legume sem graça não é ‘normal’: o segredo é textura + tempero em camadas.”
  • Sobremesa: “Pra matar vontade doce não precisa virar ‘dia do lixo’: base + upgrade pra mais saciedade.”
