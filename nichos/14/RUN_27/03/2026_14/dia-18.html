O roteiro “padrão” que faz Reels reter e bombar comentários

Identidade high-tech • fundo escuro • destaques em degradê (verde/azul/laranja)

Resumo

Modelo replicável baseado em microinteração (teste mental), virada (padrão/surpresa) e framework curto (3 palavras + regra 2/3). O final usa CTA gamificado com baixo esforço para multiplicar comentários e salvar.

Promessa (template)
Entender [conceito] em 60s + exercício aplicável hoje
Tese memorável
“Pessoas não leem mentes, leem padrões.”
Estrutura
Hook → 2 testes → virada → mito → exemplos → framework → regra → CTA
Mecânica de engajamento
Perguntas + pausa + confirmação + jogo (3 emojis)

Estrutura

  1. Hook com promessa de simplicidade + convite ao teste
  2. Teste A: 3 sinais → “o que você conclui?” (pausa)
  3. Teste B: 3 sinais → reforça o padrão (pausa)
  4. Virada: “você não adivinhou; você reconheceu um padrão”
  5. Conexão com o tema do nicho
  6. Quebra de mito (X não é; Y é)
  7. 2 exemplos em contraste (A vs B)
  8. Framework curto (3 palavras)
  9. Regra operacional (2 de 3 em cada post)
  10. CTA gamificado (comentário com 3 emojis ou 3 palavras)
Por que funciona: alterna interação → explicação → exemplo → regra, mantendo retenção e sensação de clareza.

Roteiro (plug-and-play)

Duração sugerida: 45–65s. Frases curtas. Pausas micro (0,5–1s) depois das perguntas.

[0–3s | HOOK] “Vou te explicar [CONCEITO] como se você tivesse 8 anos. Em 1 minuto você vai entender e aplicar.” [3–12s | TESTE A] “Teste rápido. Se eu te mostro: [SINAL 1], [SINAL 2], [SINAL 3]… o que isso te parece? (Respondeu?)” [12–20s | TESTE B] “Agora: [SINAL 1B], [SINAL 2B], [SINAL 3B]… e isso? (Beleza.)” [20–26s | VIRADA] “Você não adivinhou. Você só percebeu um PADRÃO.” [26–34s | TESE + CONEXÃO] “E é exatamente assim que funciona [TEMA]. As pessoas não leem mentes — elas leem padrões: um conjunto de pistas repetidas.” [34–40s | QUEBRA DE MITO] “[TEMA] não é [MITO/ALVO FALSO]. É o que as pessoas concluem do conjunto do que você mostra.” [40–50s | EXEMPLOS (A vs B)] “Exemplo: se você posta [PISTA A1], [PISTA A2], [PISTA A3], você vira ‘a pessoa [RÓTULO A]’. Mas se você posta [PISTA B1], [PISTA B2], [PISTA B3], você vira ‘a pessoa [RÓTULO B]’.” [50–58s | FRAMEWORK 3 PALAVRAS] “Exercício: escolhe 3 palavras que você quer que as pessoas sintam quando pensam em você: [PALAVRA 1], [PALAVRA 2], [PALAVRA 3].” [58–62s | REGRA 2/3] “Regra simples: todo post precisa ter pelo menos 2 dessas 3 palavras — no tema, no visual ou no jeito de falar.” [62–65s | CTA GAMIFICADO] “Agora comenta [RESULTADO/PREFERÊNCIA] usando só 3 emojis. Eu vou responder dizendo qual padrão você passa.”

Campos para substituir

  • [CONCEITO] (algo confuso no nicho)
  • [SINAIS A/B] (2 conjuntos de 3 pistas óbvias)
  • [MITO] (o erro comum que a audiência acredita)
  • [PISTAS] (comportamentos/conteúdos/indicadores)
  • [3 PALAVRAS] (atributos de marca/identidade/estratégia)
  • [CTA] (3 emojis / 3 palavras / palavra-chave no comentário)

Ritmo (cronômetro)

  • 0–3s: texto grande + promessa + convite ao teste
  • 3–20s: dois testes com pausa + “Respondeu?”
  • 20–40s: virada + tese + mito
  • 40–58s: exemplos em contraste + exercício
  • 58–65s: regra + CTA (tela final 1–2s)

Elementos visuais

  • Texto grande nos 3 primeiros segundos: “como se você tivesse 8 anos”
  • Ícones/emoji/stickers nos testes (3 itens na tela, bem grandes)
  • Destaques tipográficos em palavras-chave: PADRÃO, CONJUNTO, 3 PALAVRAS, 2/3
  • Cortes a cada 1–2 frases; zoom leve nas perguntas e na tese
  • Legenda dinâmica (karaokê) para sustentar retenção sem áudio
  • SFX leve no “reveal” (ding/pop) para marcar a virada

Dica de UX: mantenha os 3 sinais sempre alinhados (ex.: em linha) para o cérebro “ver o padrão” mais rápido.

Copy e ganchos

Hook 1 (simplicidade): “Vou te explicar [tema] como se você tivesse 8 anos.”
Hook 2 (teste rápido): “Em 5 segundos: se eu te mostro 3 sinais, o que você conclui?”
Virada: “Você não é gênio — você viu um padrão.”
  • Frase-âncora: “Tudo comunica.”
  • Quebra de crença: “Não é o que você diz que é. É o que parece quando juntam as peças.”
  • Alfinetada (sem agressividade): “Se você quer ser percebido como X, não dá pra postar como Y e esperar milagre.”
  • Fechamento com regra: “2 de 3 em todo post. Simples e executável.”

CTA

  • CTA 1 (gamificado): “Comenta seu [preferido/resultado] usando só 3 emojis.”
  • CTA 2 (diagnóstico): “Comenta suas 3 palavras e eu digo se seu conteúdo está coerente.”
  • CTA 3 (palavra-chave): “Escreve PADRÃO que eu te mando uma lista de exemplos.”

Regra: escolha um CTA por Reel para não dividir ação.

Checklist

Pré-roteiro

  • Definir [conceito] confuso porém desejado
  • Listar 2 conjuntos de 3 sinais (A e B)
  • Escrever a tese: “não leem ___, leem ___”
  • Escolher 1 mito para derrubar (X não é; Y é)

Gravação + edição

  • Texto grande até 3s + legenda dinâmica
  • Pausa após as perguntas (microinteração)
  • Destaque visual em PADRÃO e 2/3
  • Cortes a cada 1–2 frases; zoom leve nas viradas
  • Tela final com CTA por 1–2s

Variações (mesmo impacto, outros temas)

Variação 1 — “Padrões de valor” (precificação)

Hook: “Vou te explicar PREÇO como se você tivesse 8 anos.” Teste A: “Site bonito + depoimentos + garantia → caro ou barato?” Teste B: “Sem prova + sem processo + ‘me chama no direct’ → caro ou barato?” Virada: “Você não julgou o preço. Você julgou o padrão de valor.” Framework: 3 palavras (Ex.: Clareza, Prova, Processo) Regra: Todo post tem que ter 2/3 (mostra prova, explica processo, deixa claro).

Variação 2 — “Padrões de conversão” (funil)

Hook: “Vou te explicar FUNIL como se você tivesse 8 anos.” Teste A: “Promise + prova + passo seguinte → dá vontade de clicar?” Teste B: “Conteúdo aleatório + sem CTA + sem continuidade → dá vontade de clicar?” Virada: “Você não foi ‘convencido’. Você viu um padrão de decisão.” Framework: 3 palavras (Ex.: Promessa, Prova, Próximo passo) Regra: cada peça precisa ter 2/3.

Substituições prontas (biblioteca rápida)

  • “Como se você tivesse 8 anos” → “de um jeito tão simples que você aplica hoje” / “em 60 segundos”
  • 3 sinais → 3 prints / 3 falas / 3 escolhas / 3 erros / 3 hábitos
  • Mito → “não é o que você fala; é o que você repete” / “não é ferramenta; é processo”
  • 3 palavras → 3 pilares / 3 critérios / 3 promessas / 3 valores
  • CTA 3 emojis → “3 palavras” / “A ou B?” / “digita ‘QUERO’” (manter baixo esforço)
Use este template para qualquer tema: basta trocar os 2 conjuntos de 3 sinais, o mito e o framework.
