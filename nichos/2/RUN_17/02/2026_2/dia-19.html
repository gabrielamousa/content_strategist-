Reel “bye-bye” para antes/depois (transição no beat)

Roteiro reproduzível para Reels de 7–12s: 3 takes + 3 telas de texto + 1 CTA. Ideal para transformação, rebranding, encerramento de ciclo e “deixar algo pra trás”.

Resumo

Objetivo: retenção + replay

Use um áudio/frase curta que sinaliza virada (“bye-bye”) para criar curiosidade, entregar um contraste visual forte (ANTES vs DEPOIS) e fechar com CTA único. O segredo é sincronizar a transição no beat e tornar o “depois” visualmente óbvio.

  • Take 1 (0–2s): ANTES (problema/tensão) + texto curto.
  • Take 2 (2–5s): transição no “bye-bye” (mão na lente / whip-pan / virar).
  • Take 3 (5–9s): DEPOIS (resultado/prova) + benefício.
  • Fecho (9–12s): CTA: salvar ou comentar palavra-chave.

Estrutura

3 atos + loop

Ato 1 — Setup (0–2s)
Mostre o problema sem explicar demais.

Ato 2 — Virada (2–5s)
Transição exatamente no “bye-bye”.

Ato 3 — Payoff (5–9s)
Mostre o resultado de forma visual (não abstrata).

Regras de ouro

  • 1 ideia por Reel (evite explicar).
  • 3 telas de texto no máximo.
  • 1 CTA (salvar ou comentar ou seguir).
  • Loop: termine com enquadramento parecido com o início.

Roteiro (reproduzível)

7–12s

Edite este texto e use como “texto na tela” + direção de cena. O botão Copiar roteiro copia o bloco abaixo.

Salva e volta aqui quando for sua vez.” CHECK DE LOOP - Último frame com enquadramento semelhante ao primeiro para estimular replay.

Elementos visuais

Contraste + clareza
  • Contraste visível: antes frio/opaco; depois quente/vivo (ou o inverso, desde que contraste).
  • Prova na cena: em arquitetura, use planta/3D/render, canteiro, sketch, materialidade, antes/depois de obra.
  • Texto grande queimado: central e com poucas palavras (legível sem áudio).
  • Transição no beat: corte exatamente no “bye-bye”.
  • 3 takes: simplifica gravação e aumenta chance de execução.

Templates rápidos (Arquitetura)

  • Antes: “projeto travado” (muitos rascunhos/indecisão) → Depois: “partido definido” (volumetria clara).
  • Antes: “obra com retrabalho” → Depois: “detalhamento + compatibilização” (prancha/coordenação).
  • Antes: “cliente confuso” → Depois: “apresentação com narrativa” (slides/maquete eletrônica).

Copy e ganchos

1 frase, alto impacto

Ganchos (substituíveis) — troque [X] e [Y] sem perder o efeito.

  • “Adeus [X]. Olá [Y].”
  • “Eu parei de [X] quando entendi isso:”
  • “Se você ainda faz [X], isso explica [dor].”
  • “O ‘bye-bye’ que mudou meu [processo/resultado].”

Substituições com o mesmo impacto

  • [X] (o que você deixa): hábito ruim, erro técnico, crença limitante, rotina, ferramenta, tipo de cliente, padrão estético.
  • [Y] (o que entra): método, padrão de qualidade, checklist, template, disciplina, posicionamento, princípio de projeto.
  • Prova visual: número (antes/depois), tela (timeline), documento (prancha), transformação (ambiente/obra), comparação (render vs executado).

CTA

Escolha 1
  • Para gerar DM: “Comenta PIE que eu te mando o checklist.”
  • Para alcance/SEO: “Comenta: qual [X] você está deixando?”
  • Para retenção: “Salva pra usar quando bater vontade de voltar pro [X].”
  • Para série: “Me segue pra parte 2: como eu fiz o [Y].”

Regra: não misture 2 CTAs no mesmo Reel. Quer DM? então não peça salvar no mesmo vídeo (deixe salvar para a legenda, se necessário).

Checklist de criação

Execução sem travar

Pré-produção (10 min)

  • Defina [X] em até 3 palavras.
  • Defina [Y] em 5–7 palavras + 1 benefício.
  • Escolha prova visual do depois (o que a câmera vai mostrar).
  • Selecione 1 transição (mão na lente / whip pan / virar).
  • Escreva 3 telas de texto (máx. 6–8 palavras cada).

Gravação (15 min)

  • Take ANTES: 1–2s (close + luz fria).
  • Take TRANSIÇÃO: marque o beat do “bye-bye”.
  • Take DEPOIS: 2–4s (plano aberto + prova).
  • Grave 2 variações do DEPOIS (muda o plano/ângulo).

Edição/Publicação (15–25 min)

  • Corte no beat exato.
  • Texto grande e central.
  • Color grade com contraste antes/depois.
  • Finalize com loop (frame parecido com o início).
  • Legenda 2–4 linhas + 3–6 hashtags específicas.

Variações

Adapta para qualquer tema

Variação 1 — “Erro comum”
Troque o drama pessoal por um erro técnico.

  • Antes (texto): “Pare de fazer [erro].”
  • Depois (texto): “Faça [correção] e ganhe [benefício].”

Variação 2 — “Mito vs Verdade”
Use “bye-bye” para matar um mito.

  • Antes: “Mito: [crença].”
  • Depois: “Verdade: [princípio].”

Variação 3 — “Bastidores”
Antes = caos; depois = processo.

  • Antes: “Ninguém vê quando [dor].”
  • Depois: “O que eu faço é: [passo 1–2].”

Variação 4 — Arquitetura (pronto para usar)

  • X: “Detalhe mal resolvido” → Y: “Detalhamento + compatibilização”
  • X: “Apresentação confusa” → Y: “Narrativa + comparativos (antes/depois)”
  • X: “Obra no improviso” → Y: “Checklist de obra + fotos padrão”

Dica: em arquitetura, o “depois” precisa ser concreto: prancha, detalhe, render, foto de obra, moodboard final, cronograma ou comparação executado vs projeto.
