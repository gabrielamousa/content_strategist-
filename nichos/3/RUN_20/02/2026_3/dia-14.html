Quando ser “a única” parece força (mas te esgota)

Roteiro curto com padrão setup → quebra + tríade. Perfeito para falar de imagem pessoal como “papel” que você sustenta sozinha (a que resolve, a impecável, a forte) — e como isso cobra energia, estilo e presença.

Ir para checklist

Resumo

Objetivo
  • Gerar identificação rápida com o peso de sustentar um “papel” através da imagem.
  • Transformar elogio em alerta: “parece força” → “é sobrecarga”.
  • Estimular comentários e salvamentos com a palavra-âncora ÚNICA.
Promessa
  • Você vai perceber onde sua imagem virou “função” (resolver, agradar, ser impecável) e por que isso te drena.
  • Você sai com um termo para nomear o padrão e iniciar a conversa (consigo/relacionamentos/trabalho).

Duração ideal: 6–10s. 3 repetições rápidas. Legenda grande e central.

Estrutura

  • Hook: “Essa aqui é uma [ARQUÉTIPO], mas quando você é a única [FUNÇÃO INVISÍVEL], fica assim.”
  • Tríade: 3 arquétipos diferentes + mesma cadência.
  • Quebra: postura confiante → queda de energia no “fica assim”.
  • Fechamento opcional (1 frase): “Se você é a única [função], isso não é força — é falta de reciprocidade.”
  • CTA: comentário (“única em ____”) + salvar.

Roteiro

[CENA 0 — 0:00–0:01 | Close, postura firme] TEXTO NA TELA: “Isso parece força.” FALA (opcional): “Isso parece força…” [CENA 1 — 0:01–0:03 | Arquétipo 1] TEXTO NA TELA: “Essa é uma IMPONENTE” FALA: “Essa aqui é uma imponente…” TEXTO NA TELA (linha 2): “mas quando você é a ÚNICA que mantém a postura…” FALA: “…mas quando você é a única que mantém a postura…” TEXTO NA TELA (linha 3): “fica assim.” AÇÃO: ombros caem, exala, olhar perde brilho. [CENA 2 — 0:03–0:06 | Arquétipo 2] TEXTO NA TELA: “Essa é uma IMPECÁVEL” FALA: “Essa aqui é uma impecável…” TEXTO NA TELA (linha 2): “mas quando você é a ÚNICA que ‘dá conta de tudo’…” FALA: “…mas quando você é a única que dá conta de tudo…” TEXTO NA TELA (linha 3): “fica assim.” AÇÃO: mão na testa / micro-olhar de irritação / cansaço. [CENA 3 — 0:06–0:09 | Arquétipo 3] TEXTO NA TELA: “Essa é uma ELEGANTE” FALA: “Essa aqui é uma elegante…” TEXTO NA TELA (linha 2): “mas quando você é a ÚNICA que sustenta o clima…” FALA: “…mas quando você é a única que sustenta o clima…” TEXTO NA TELA (linha 3): “fica assim.” AÇÃO: sorriso some, olhar pro lado, desanima. [FECHO — 0:09–0:10 | 1 frase opcional] TEXTO NA TELA: “ÚNICA ≠ forte. ÚNICA = sobrecarga.” FALA: “Se você é a única, isso não é força. É sobrecarga.”

Dica: troque os arquétipos para combinar com sua audiência (executiva, mãe, empreendedora, etc.). O essencial é manter a cadência e a palavra “ÚNICA” sempre com a mesma cor.

Elementos visuais

  • Look: comece mais “polida” (blazer/colar/arrumada), e no “fica assim” bagunce levemente (soltar um botão, tirar o salto, mexer no cabelo) — metáfora visual do desgaste.
  • Enquadramento: close no rosto + zoom-in progressivo a cada repetição.
  • Legenda: grande, central, com ÚNICA sempre na mesma cor (ex.: amarelo/laranja).
  • Edição: cortes secos por frase; beat/whoosh suave no “fica assim”.
  • Expressões: 3 “fica assim” diferentes (cansaço, irritação, vazio) para dar variedade sem mudar o texto.

Copy e ganchos

Ganchos (escolha 1):

  • “Isso parece autoestima… até você ser a única que sustenta.”
  • “Sua imagem pode virar um papel. E papel cansa.”
  • “Você não é ‘forte’. Você tá sozinha nisso.”

Legenda pronta (curta):

Tem elogio que vira armadilha: ser “a imponente”, “a impecável”, “a elegante”… Mas quando você é a ÚNICA que mantém a postura, resolve tudo e sustenta o clima, o corpo cobra. Comenta: “única em ____”. Salva pra lembrar: virtude sem reciprocidade vira peso.

CTA

  • Para comentários: “Você tá sendo a única em quê? Comenta: ‘única em ____’.”
  • Para salvamento: “Salva pra lembrar: ÚNICA não é força, é sobrecarga.”
  • Para compartilhamento: “Envia pra alguém que vive sendo ‘a que dá conta’.”

Checklist

  • Tema definido: “o custo de sustentar sozinha um papel através da imagem”.
  • Escolher 3 arquétipos (status/identidade) alinhados ao seu público.
  • Definir 1 função invisível para cada arquétipo (ex.: manter postura, resolver, agradar, sustentar clima, ser referência).
  • Escrever 3 blocos idênticos no formato: “Essa é uma [A] / mas quando você é a única [B] / fica assim.”
  • Planejar 3 expressões corporais diferentes para o “fica assim”.
  • Gravar em 9:16 com luz frontal, fundo simples e enquadramento no busto/rosto.
  • Editar com cortes secos + zoom progressivo; inserir ênfase sonora no “fica assim”.
  • Legendar grande e central; destacar ÚNICA sempre com a mesma cor.
  • Escolher 1 CTA (comentário OU salvar/compartilhar) e fixar nos 2 últimos segundos.

Variações

Variação A (trabalho / liderança):

“Essa é a COMPETENTE. Mas quando você é a ÚNICA que resolve crise… fica assim.” “Essa é a ESTRATÉGICA. Mas quando você é a ÚNICA que pensa no todo… fica assim.” “Essa é a PROFISSIONAL. Mas quando você é a ÚNICA que segura a barra… fica assim.”

Variação B (relacionamentos / vida social):

“Essa é a TRANQUILA. Mas quando você é a ÚNICA que evita conflito… fica assim.” “Essa é a CARINHOSA. Mas quando você é a ÚNICA que puxa assunto… fica assim.” “Essa é a MADURA. Mas quando você é a ÚNICA que cede sempre… fica assim.”

Mantenha sempre: identidade admirável → “mas quando você é a ÚNICA…” → queda de energia. A repetição é o motor.
