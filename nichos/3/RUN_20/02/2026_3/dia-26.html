5 peças para investir e usar o ano inteiro (com 1 truque)

Post original: https://www.instagram.com/p/DUoqilBjkHq/ (use como referência de ritmo/edição)

Resumo

Ideia-base: Reel em formato de lista (5 itens) com promessa ampla (“usar o ano inteiro”) + um twist unificador (ex.: cinto/terceira peça/tênis) + prova rápida ("eu comprei/uso muito").

Promessa
Investir agora, usar o ano inteiro
Mecânica de retenção
Lista 1/5 → 5/5 + cortes rápidos
CTA
Comente 1–5 + “LINKS”
Custo por uso Versatilidade por estação Guia de compras Autoridade leve

Estrutura (reproduzível)

  • 0–2s: Hook direto (promessa + twist).
  • 2–22s: Itens 1–5 (3–4s cada), sempre com o mesmo mini-roteiro.
  • 22–28s: Pergunta de identidade ("qual é sua cara?") + CTA de comentários.
  • 28–32s: Assinatura (quem você é + o que você resolve).

Regra de ouro: 1 ideia forte por item + repetição de padrão (calor/frio) para o cérebro “seguir a trilha” sem esforço.

Roteiro (pronto para gravar)

Pré-set: escolha 5 peças (ou 3, se quiser mais rápido) e 1 twist (cinto / terceira peça / tênis / lenço / acessórios dourados / monocromático).

Elementos visuais

  • Texto inicial grande (1 linha): promessa + twist.
  • Contador fixo: 1/5… 5/5 (canto superior).
  • Match cut: mesma peça em 2 versões (calor → frio) para provar versatilidade.
  • B-roll 0,5–1s: textura/tecido/cinto/detalhe quando você citar o benefício.
  • Ritmo: jump cuts a cada 1–2 frases (sem contexto longo).
  • Tela final: pergunta + “comenta LINKS”.

Copy e ganchos

  • Hook A: “5 [peças] pra investir agora e usar o ano inteiro (com [twist])”
  • Hook B: “Se você quer comprar menos e usar mais: 5 [peças] que atravessam as estações.”
  • Hook C (dor): “Pare de comprar ‘uma vez só’: 5 [peças] que rendem looks no calor e no frio.”
  • Frases de alívio: “sem esforço”, “resolve o armário”, “custo por uso lá embaixo”.
  • Microprova: “eu acabei de comprar / eu uso muito / virou minha peça curinga”.

CTA

  • CTA primário (comentários): “Qual é mais a sua cara? Comenta 1–5.”
  • CTA com recompensa: “Quer os links/referências? Comenta ‘LINKS’ que eu te envio.”
  • CTA extra (baixo atrito): “Salva pra consultar quando for comprar/montar mala.”

Checklist de criação

  • Promessa: ampla e desejada (ex.: “usar o ano inteiro”, “do trabalho ao fim de semana”).
  • Lista: 5 itens (ou 3 para versão curta).
  • Twist: 1 regra visual que une tudo (cinto / terceira peça / mesma bolsa / mesma paleta).
  • Por item (sempre igual): nome → por que vale → calor → frio.
  • Prova: inclua 1 frase real (“eu comprei/uso muito”).
  • Edição: contador 1/5 + cortes rápidos + match cut calor/frio.
  • Texto na tela: frases curtas, escaneáveis; repita “calor/frio”.
  • CTA: pergunta de identidade + “LINKS” (responda rápido para manter volume).
  • Publicação: fixe um comentário perguntando “qual é sua cara? 1–5”.

Dica de reuso: mantenha a mesma estrutura e troque apenas (1) a promessa, (2) os 5 itens, (3) o twist e (4) o “calor/frio” por outro contraste.

Variações (adaptações com o mesmo impacto)

1) Trocas de “contraste” (no lugar de verão/inverno)

  • Trabalho → fim de semana (formal x casual)
  • Dia → noite (mudando acessórios/terceira peça)
  • Básico → elegante (mesma peça, styling diferente)
  • Minimalista → fashion (mesma base, toque de tendência)
  • Viagem → rotina (mala inteligente)

2) Trocas de “twist” (regra visual unificadora)

  • Com cinto (marcar cintura / proporção)
  • Com terceira peça (blazer, jaqueta, colete, cardigan)
  • Com um mesmo sapato (tênis branco, loafer, bota)
  • Com uma paleta (neutros + 1 cor destaque)
  • Com acessórios (colar statement, dourado, lenço)

3) Exemplos prontos de temas (mesma estrutura)

  • “5 sapatos que valem o investimento e funcionam o ano inteiro (com meias/sem meias)”
  • “5 peças de trabalho que viram looks de fim de semana (com troca de acessórios)”
  • “5 opções até R$X que parecem caras (com 1 truque de styling)”
  • “5 peças para plus size que alongam sem esforço (com terceira peça)”
  • “5 looks para viajar leve e repetir sem ninguém perceber (com paleta cápsula)”

4) Substituições de CTA (você escolhe)

  • “Comenta ‘LISTA’ que eu te mando as referências em DM.”
  • “Quer versão ‘até R$X’? Comenta ‘R$X’.”
  • “Qual peça você mais usa hoje? Comenta que eu sugiro 2 combinações.”
  • “Salva e me marca quando montar um look com isso.”
