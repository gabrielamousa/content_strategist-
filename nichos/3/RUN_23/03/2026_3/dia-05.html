Carrossel infalível: 10 slides para consultoria de imagem

Roteiro reproduzível para posts educativos com retenção (hook → dor → mito → passos → checklist → CTA). Ajuste apenas o tema: cores, proporção, dress code, cápsula, compras, acessórios.

Ver post original Objetivo: educar + gerar saves CTA sugerido: comentar palavra-chave / DM

Resumo

O que é: um template de carrossel (8–10 slides) que você replica para qualquer assunto de imagem pessoal.

Por que funciona: tem promessa clara, quebra de crença, aplicação prática e termina com checklist “salvável”.

1 ideia por slide Frases curtas (até 12 palavras) Loop (“no próximo slide…”) 2–3x Exemplos concretos

Copiar roteiro

Dica: depois de copiar, ajuste apenas as variáveis entre [COLCHETES].

Estrutura

Sequência (padrão reproduzível)

  • Slide 1: capa com hook + promessa (benefício + objeção).
  • Slide 2: empatia + por que importa (consequência de continuar errando).
  • Slide 3: erro/mito principal (quebra de crença).
  • Slide 4: ideia-chave (critério simples para decisão).
  • Slides 5–7: 3 passos práticos com microexemplos.
  • Slide 8: checklist “salvável”.
  • Slide 9: aplicação em cenários reais (trabalho, casual, evento).
  • Slide 10: CTA (comentário/DM) + CTA secundária (salvar/enviar).

Regras de copy (para manter consistência)

  • 1 ideia por slide e no máximo 2 frases (exceto checklist).
  • Frases curtas, linguagem direta, verbos no imperativo (use, prefira, evite).
  • Inclua 2–3 loops (“no próximo slide…”) para retenção.

Roteiro

Roteiro-mestre (preenchível)

Use como “esqueleto” textual para qualquer tema.

  • “Se você sente que [dor], provavelmente está [erro].”
  • “O ponto não é [X]. É [Y].”
  • “Use este critério: [regra].”
  • “Aplique assim: [passo 1], [passo 2], [passo 3].”
  • “Resumo: [checklist].”
  • “Quer que eu adapte ao seu caso? [CTA].”

Slides sugeridos (1..10)

  1. Capa: “Como [resultado] sem [objeção]” + “salve para usar”.
  2. Empatia: “Se você [sintoma], não é falta de estilo. É falta de método.”
  3. Erro: “O erro que te faz gastar e ainda se sentir ‘nada a ver’.”
  4. Regra: “Regra de ouro: [critério] antes de escolher a peça.”
  5. Passo 1: [orientação] + exemplo real.
  6. Passo 2: [orientação] + exemplo real.
  7. Passo 3: [orientação] + exemplo real.
  8. Checklist: 4–6 itens com caixa de seleção.
  9. Aplicação: “Se seu objetivo é X, faça Y.” (2–3 cenários).
  10. CTA: comentar palavra-chave/DM + salvar/enviar.

Elementos visuais

Direção de arte (identidade consistente)

  • Fundo: escuro (grafite/azul noite) com textura sutil.
  • Cards: blocos com borda suave e bastante respiro.
  • Destaques: palavras-chave com faixa em degradê (verde → azul → laranja).
  • Elementos fixos: numeração “3/10”, barra fina de destaque, @ discreto no rodapé.

Tipografia (simples e premium)

  • 2 fontes no máximo: uma para títulos (bold) e outra para corpo (regular).
  • Hierarquia: título grande (1–2 linhas), subtítulo menor, bullets curtos.

Copy e ganchos

Modelos de título (até 70 caracteres)

  • “Como parecer mais elegante sem comprar roupa nova”
  • “Você está errando no [tema] (faça isso)”
  • “Guia rápido: [tema] em 3 passos”
  • “Checklist para [resultado] (salve)”

Ganchos por slide (para retenção)

  • “O problema não é a peça — é o critério.”
  • “Você só precisa de 1 ajuste para mudar a leitura.”
  • “No próximo slide: o detalhe que quase ninguém observa.”
  • “Se você faz isso, seu look fica caro sem esforço.”

CTA

Escolha 1 primária + 1 secundária

  • Primária (comentário): “Comente ‘IMAGEM’ que eu te mando o checklist.”
  • Primária (DM): “Me chama no DM com ‘ANÁLISE’ e me diga: objetivo + ocasião.”
  • Secundária (saves): “Salve para consultar na hora de se vestir.”
  • Secundária (share): “Envie para alguém que sempre trava na hora do look.”

Sugestão de baixa fricção: peça uma palavra-chave única e responda com mensagem padrão (guia + oferta do serviço).

Checklist

Checklist de criação (produção rápida)

  • Definir tema: [cores | proporção | dress code | cápsula | compras | acessórios].
  • Definir promessa: [resultado] sem [objeção].
  • Escolher 1 erro/mito para quebrar no slide 3.
  • Escrever 3 passos com 1 exemplo cada.
  • Montar checklist com 4–6 itens “salváveis”.
  • Adicionar CTA: palavra-chave + salvar/enviar.
  • Revisar densidade: 1 ideia/slide, frases curtas, contraste alto.
  • Publicar: capa forte + legenda curta com reforço do CTA.

Variações

Versão curta (6–7 slides)

  • Capa → Contexto → 3 passos → Checklist → CTA

Versão “Mito vs Verdade” (8 slides)

  • Capa → por que importa → 3 mitos (1 por slide) → 3 verdades (1 por slide) → CTA

Versão “Erro comum” (9 slides)

  • Capa → erro → consequência → regra → o que fazer (3 slides) → checklist → CTA
