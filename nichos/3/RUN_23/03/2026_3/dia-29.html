5 peças versáteis que resolvem looks o ano inteiro

Roteiro reproduzível baseado em uma estrutura vencedora: promessa concreta + lista + repetição (mini-bloco) + prova pessoal + CTA duplo para comentários.

Abrir post original

Resumo O que faz funcionar

  • Promessa específica e imediata: lista curta com benefício claro (investir melhor / usar o ano todo).
  • Recompensa: “com links / com guia” reduz atrito e aumenta comentários.
  • Padrão repetido: o mesmo mini-roteiro em cada item dá ritmo e retenção.
  • Prova e proximidade: “eu uso / eu comprei” aumenta confiança.
  • Resolve uma dor real: medo de comprar e usar pouco (custo por uso + versatilidade).
  • CTA duplo: 1) preferência (identidade), 2) palavra-chave (envio de recurso).
Objetivo
Comentários + salvamentos
Duração sugerida
45–60s
Ritmo
1 item a cada 7–10s

Estrutura Template fixo

  1. Hook (0–2s): número + promessa + recompensa.
  2. Itens 1–5: repetir mini-bloco (o que é → por que vale → verão → inverno → efeito).
  3. Fechamento: pergunta de preferência + palavra-chave + assinatura.
Regra de ouro: mantenha a mesma “moldura” e troque só o conteúdo (itens/benefícios). Isso facilita replicar em qualquer tema.

No frio: [como usar]. Efeito: [efeito].” FECHAMENTO (últimos 8–12s) “Qual é mais a sua cara: 1, 2, 3, 4 ou 5?” “Quer [RECOMPENSA]? Recompensa (escolha 1): links / lista de compras / checklist de combinações / guia PDF / "como adaptar ao seu corpo".
Palavra-chave (escolha 1): “LINKS”, “GUIA”, “LISTA”, “ARMÁRIO”, “QUERO”.

Elementos visuais Edição e tela

  • Contador fixo no canto: 1/5, 2/5…
  • Texto curto por bloco: “Vale porque: …” / “Verão:” / “Inverno:” / “Efeito: …”
  • B-roll alternando: look no corpo ↔ print/foto do item ↔ close do tecido/caimento.
  • Punch-in quando falar o benefício principal (“não marca”, “alongar”, “refina”).
  • Transição repetida (sempre a mesma) entre itens para criar padrão.
  • Tela final com CTA grande: “Comente ‘GUIA’”.
Dica de copy na tela: mantenha cada linha com no máximo 6–8 palavras para leitura rápida.

Copy e ganchos Modelos substituíveis

Hooks (escolha 1):
  • “5 peças para investir agora e usar o ano inteiro (com [recompensa]).”
  • “Se você quer [benefício], começa por essas 5.”
  • “Eu escolheria essas 5 se fosse montar um armário enxuto do zero.”
  • “Quer parecer mais arrumada sem esforço? 5 escolhas.”
Frases de retenção (espalhe ao longo):
  • “Isso atravessa as estações.”
  • “O caimento faz todo o trabalho.”
  • “Você usa muito mais do que imagina.”
  • “Dá acabamento sem pesar o look.”
  • “Custo por uso baixíssimo.”
Substituições que mantêm o impacto: troque “peças” por “cores”, “modelagens”, “erros”, “truques”, “combinações”, “acessórios”; troque “verão/inverno” por “trabalho/fim de semana”, “dia/noite”, “casual/arrumado”.

CTA Duplo (alta interação)

  • Preferência (identidade): “Qual é mais a sua cara? 1, 2, 3, 4 ou 5?”
  • Recurso (palavra-chave): “Quer [recompensa]? Comenta ‘[PALAVRA]’ que eu te envio.”
  • Personalização (opcional): “Quer versão para [contexto]? Comenta ‘QUERO + [contexto]’.”
Se você não envia DM: troque por “eu deixo fixado/ eu libero nos stories/ eu mando por broadcast”.

Checklist Produção → gravação → edição

  • Defina 1 promessa
    Ex.: “usar o ano inteiro”, “parecer mais elegante”, “alongar”, “refinar”, “evitar comprar errado”.
  • Selecione 5 itens
    Peças/looks/erros/acertos. Misture 3 “básicos que resolvem” + 2 “toques de estilo”.
  • Para cada item, preencha o mini-bloco
    O que é + por que vale (1 motivo técnico) + como usar no cenário A + cenário B + efeito final.
  • Escolha a recompensa
    Links / lista / guia / checklist / “10 combinações prontas”.
  • Grave com cadência
    0–2s hook; 7–10s por item; feche com 2 CTAs em 8–12s.
  • Edite com padrão visual
    Contador 1/5; rótulos “Verão/Inverno” (ou equivalentes); tela final com palavra-chave.
  • Publique + responda rápido
    Primeira hora: priorize respostas aos comentários que contenham a palavra-chave.

Variações Mesma estrutura, temas diferentes

  • “5 terceiras peças que refinam qualquer look” (blazer, jaqueta, colete, cardigan, trench).
  • “5 erros que te envelhecem (e o que fazer no lugar)” (mantém lista + mini-bloco com “troca”).
  • “5 combinações para parecer mais alta” (monocromia, cintura alta, decote V, ponta fina, barra certa).
  • “5 cores que valorizam [subtom]” (como usar no dia/noite).
  • “5 peças para [contexto]” (trabalho, viagem, eventos, armário cápsula, pós-parto).
Mapa de adaptação rápida: [NÚMERO] + [CATEGORIA] + [BENEFÍCIO] + “sem esforço” + [RECOMPENSA].
