Nicho: Marketing e Posicionamento
Dia: 5
Formato: Reels
Pilar: Passo a passo

Transforme qualquer nicho em marca premium com 1 template

Execute o mesmo padrão do Reel: objeto/serviço comum → tese (manifesto) → naming + estética → 3 versões → experiência de lançamento → prova de método → filtro + CTA.

Identidade + experiência + sistema
Ritmo: 1 ideia a cada 2–3s

Resumo

Objetivo: desejo + autoridade

O que você vai criar

Um Reel que “re-embala” um serviço/produto comum como ritual (posicionamento premium), com 3 versões e uma ativação memorável — finalizando com método + CTA.

Por que funciona

Quebra padrão (imaginação), entrega tese polarizante, usa nomes próprios (memória), “escaneabilidade” (estrutura 3 atos) e detalhes específicos (credibilidade).

Gancho
Objeto comum + mundo paralelo
Motor
3 versões com estética própria
Fechamento
Método + filtro + CTA direto

Checklist rápido (antes de gravar)

5–10 min
Escolha o “comum”: produto/serviço banal do seu público (ex.: consultoria, clínica, curso, contabilidade, imobiliária, app).
Defina o conceito elevado: ritual / status / performance / liberdade / tranquilidade.
Escreva o manifesto (contraste): “não deveria ser X, deveria ser Y”.
Crie 3 versões com nomes memoráveis + 3 elementos + 3 benefícios curtos.
Ativação: descreva um evento/experiência com 2–3 detalhes concretos.
CTA conectado: “quer que eu faça isso no seu negócio? comenta/manda palavra-chave”.

Estrutura (sequência reproduzível)

Template em 8 blocos

A) Premissa imaginativa

0–2s

“Se eu fosse criar um(a) [X comum], eu faria assim…”

B) Manifesto (tese polarizante)

2–5s

“Porque [ação] não deveria ser [mecânico/barato/chato], deveria ser [ritual/sagrado/premium].”

C) Naming + estética

5–8s

“Nome: [Marca/Oferta]. Visual: [1 detalhe de design] + [1 textura].”

D) Linha em 3 versões (coração)

8–25s

“São 3 versões: [V1], [V2], [V3].”
Para cada versão: nome + 3 elementos + 3 benefícios (tudo curto e nomeável).

E) Elevação de valor

25–28s

“Não é só [função]. É [identidade/estilo de vida/transformação].”

F) Ativação de lançamento

28–34s

“Lançamento: [evento] com [detalhe 1], [detalhe 2], [detalhe 3].”

G) Virada de método + autoridade

34–40s

“É por isso que posicionamento funciona em qualquer segmento: você não vende [coisa], você vende [significado + experiência + sistema].”

H) Filtro + CTA

40–45s

“Se você quer [resultado], comenta ‘RITUAL’/manda ‘POSICIONAMENTO’. (Não é pra quem quer só ‘postar todo dia’ sem estratégia.)”

Regra de ritmo: a cada 2–3s, uma “coisa nomeável” na tela (nome, benefício, ingrediente/elemento, cena). Isso aumenta retenção e comentários.

Elementos visuais

Edição & direção

Edição

Jump cuts por frase. Texto na tela só com palavras-chave (nomes + benefícios). Velocidade 1.05–1.15x se ficar natural.

Transições por estética

V1 (manhã): verde/claridade • V2 (tarde): laranja/energia • V3 (noite): azul/roxo/calma. Troque luz, fundo ou overlay de cor para “marcar” cada versão.

  • Detalhes específicos (textura/ingrediente/feature) vendem credibilidade: “tubo transparente”, “tampa magnética”, “interface sem menu”, “sala imersiva”.
  • B-roll/overlays: café/matcha/água/luz do dia; telas, UI, grid, embalagens (se não tiver, use imagens stock rápidas).
  • SFX leve de “pop” quando aparecer cada nome (opcional).

Copy e ganchos

Frases para abrir loops

Hooks (escolha 1)

1) “Se eu fosse criar um(a) [X], eu faria assim…”
2) “[Ação do mercado] não deveria ser [X], deveria ser [Y].”
3) “Eu transformaria [categoria] em um ritual.”

Manifestos (contraste)

“Não é sobre [tarefa]. É sobre [identidade].”
“Não é mais um [serviço]. É um sistema.”
“O comum vira premium quando você desenha a experiência.”

Texto na tela (11 cards rápidos)

1. “Se eu criasse [categoria], seria assim:”
2. “Não é mecânico. É ritual.”
3. “Nome: [BRAND/OFFER]”
4. “3 versões. 3 momentos.”
5. “1) [V1] → [3 benefícios]”
6. “2) [V2] → [3 benefícios]”
7. “3) [V3] → [3 benefícios]”
8. “Não é só [função].”
9. “É [identidade].”
10. “Isso é posicionamento.”
11. “Comenta ‘RITUAL’ / ‘POSICIONAMENTO’”

CTA

Direto + conectado ao vídeo
  • Comentário (alto alcance): “Comenta RITUAL que eu te mando o template com exemplos pro seu segmento.”
  • Direct (qualificação): “Me manda POSICIONAMENTO + seu nicho + ticket.”
  • Oferta (próximo passo): “Quer que eu desenhe suas 3 versões + naming + ativação? Link na bio.”
  • Filtro (polarização leve): “Se você quer só ‘postar mais’, pula. Se quer marca premium, vem.”

Checklist (criação completa)

Pré-roteiro • Roteiro • Edição

Pré-roteiro

• Escolher o “comum” do público
• Escolher o conceito elevado (ritual/status/performance)
• Escrever 1 manifesto (contraste)

Roteiro (garantir os blocos)

• Hook “Se eu fosse criar…”
• Manifesto em 1–2 frases
• Nome da marca/oferta + porquê
• 1 detalhe visual (design/forma/textura)
• 3 versões (nome + 3 elementos + 3 benefícios)
• Elevação: “não é só… é…”
• Ativação: evento/experiência com 2–3 detalhes
• Método: “isso prova que…”
• Filtro + CTA

Edição

• Corte por frase (sem pausas longas)
• Texto na tela com palavras-chave
• Mudança de cor por versão
• CTA nos últimos 2s

Variações (para não repetir igual)

3 modelos prontos

Variação 1 — Serviços (agência/consultoria)

3 níveis

V1 Start: clareza + diagnóstico + mapa • V2 Scale: funil + conteúdo + oferta • V3 Icon: posicionamento + PR + colabs.
Ativação: “sala de estratégia” ao vivo com entrega em 48h.

Variação 2 — Infoproduto

3 rotinas

Manhã: foco & consistência • Tarde: prática guiada • Noite: revisão & plano. Ativação: “desafio 7 dias” com checkpoint e ranking (sem virar bagunça).

Variação 3 — Negócio local

experiência

Transforme o “produto” em ritual: boas-vindas, assinatura sensorial, final memorável. Ativação: “noite temática” + brinde + foto spot + cupom com nome da versão.
